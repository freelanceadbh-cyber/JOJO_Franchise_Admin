JoJo Ice Creams Franchise Management PortalProject Workflow & Software Requirements Document
Project Name: JoJo Ice Creams Franchise Management PortalCurrent Version: Static HTML promotional website.Proposed Version: Complete web application with franchise ordering, payment processing, invoice generation, dashboards, order management, and admin portal.
1. Project Objective
Transform the current promotional website into a complete Franchise Management System that allows franchise owners to order products, pay online, track orders, download invoices, and communicate with headquarters while providing administrators with tools to manage products, orders, payments, franchises, and analytics.
2. User Roles
Visitor• View public pages and product information.Franchise Owner• Login/logout• Dashboard• Browse products• Place orders• Pay online• Download invoices• Track orders• View order history• Send messages• Manage profileBrand Owner (Admin)• Manage franchise accounts• Manage products• Manage orders• Manage payments• View reports• Reply to messages• Dashboard & analytics• Manage profile
